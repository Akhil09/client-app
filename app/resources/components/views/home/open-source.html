O que é Open Source? Segundo a Wikipédia, o Software de código aberto (em português) é um software de computador com o seu código fonte disponibilizado e licenciado sob uma licença de código aberto no qual o direito autoral fornece o direito de estudar, modificar e distribuir o software de graça para qualquer um e para qualquer finalidade.
Mas porque o Zueira Futebol Clube é um projeto Open Source? Quando você expõe suas virtudes e fraquezas ao ambiente colaborativo é muito mais simples e rápido de crescer, do que se estivéssemos trancados em uma sala!
